ChatGPT to PDF - 使用示例

这个页面展示了 ChatGPT to PDF 油猴脚本的使用效果和界面截图。通过这个脚本，您可以轻松地将 ChatGPT 的对话导出为 PDF 文件，保留所有格式和样式。

界面展示

安装脚本后，ChatGPT 页面右下角会出现两个按钮：

导出PDF 设置

点击"导出PDF"按钮，会立即将当前对话导出为PDF文件。点击"设置"按钮，会打开设置面板：

PDF导出设置

导出效果

导出的PDF文件会保留ChatGPT对话的所有格式和样式，包括：

代码块

保留代码格式和语法高亮，自动换行，不会被截断。

function helloWorld() {
  console.log("Hello, world!");
}

表格

表格结构完整，边框和背景色保持一致。

图片

图片完整显示，大小适应页面宽度，保持清晰度。

Markdown格式

标题、列表、引用块等Markdown格式元素都能正确显示。

使用步骤

  1. 安装Tampermonkey浏览器扩展
  2. 安装ChatGPT to PDF脚本
  3. 访问ChatGPT网站并进行对话
  4. 点击右下角的"导出PDF"按钮
  5. 等待PDF生成并自动下载
注意： 对于非常长的对话，可能需要一些时间来生成PDF。如果遇到问题，可以尝试刷新页面后重试。

自定义选项

通过设置面板，您可以自定义以下选项：

  • 文件名：设置导出PDF的文件名
  • 页面大小：选择A4、Letter或Legal格式
  • 方向：选择纵向或横向
  • 包含时间戳：是否在PDF中包含导出时间
  • 包含标题：是否在PDF中包含"ChatGPT 对话记录"标题

所有设置会自动保存到浏览器的本地存储中，下次使用时自动加载。
